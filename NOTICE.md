# Сторонние ассеты и лицензии

## Графика офиса (тайлы, мебель, собаки-агенты, вывеска)
Пиксель-арт офиса (`app/public/assets/tilesets/*.png`) — **собственная графика автора**.
Часть ассетов генерируется процедурно скриптом `tools/gen_office_tiles.mjs`
(`node tools/gen_office_tiles.mjs`). Распространяется по лицензии репозитория (MIT).

## Персонажи (LimeZu — Modern Interiors, FREE) — НЕ входят в репозиторий
Для спрайтов людей (Adam/Alex/Amelia/Bob) в разработке использовалась бесплатная версия
LimeZu «Modern Interiors». **Лицензия FREE-версии запрещает перераспространение файлов**,
поэтому оригинальные спрайты LimeZu **исключены** из этого репозитория.

Чтобы запустить с «родными» персонажами:
1. Скачайте бесплатную версию LimeZu (itch.io): https://limezu.itch.io/moderninteriors
2. Положите спрайты в `app/public/assets/limezu/Characters_free/` с именами вида
   `Adam_idle_16x16.png`, `Adam_run_16x16.png`, `Adam_sit_16x16.png` (аналогично Alex/Amelia/Bob).
3. Папка `app/public/assets/limezu/` в `.gitignore` — обратно в репозиторий не попадёт.

Без этих файлов приложение запускается и работает (офис, мебель, собаки-агенты, панель,
анимации), но человеческие персонажи будут отображаться плейсхолдером.
Для коммерческого использования купите полную версию LimeZu либо замените графику на свою/CC0.

## Данные 1С / Bitrix
Реальные данные (`app/public/assets/data/real.json`) и креды (`scripts/.env`) **не коммитятся**
(см. `.gitignore`). Снапшот генерируется скриптом `scripts/build_snapshot.py` из вашей 1С.
В репозитории нет ни данных конкретного бизнеса, ни паролей, ни адресов серверов.
